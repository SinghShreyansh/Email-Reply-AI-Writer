# Privacy Policy for Email Writer Assitant

**Last Updated:** February 16, 2026

## 1. Introduction
This Chrome extension (“Email Writer Assitant”) is designed for private use and assists users by generating replies based on the content of emails or text selected by the user.

## 2. Data We Collect
The Extension **does not collect or store any personal information**.

The only data processed is:
- Text the user selects or inputs voluntarily  
This text is temporarily sent to our backend API solely to generate a reply.

## 3. How Your Data Is Used
- Data is used **only** to generate a contextual reply.
- No personal data is saved.
- No data is shared with third parties.
- No tracking, analytics, or logging of user content occurs.

## 4. Storage
The Extension may use Chrome storage only for local configuration (e.g., user settings). This data:
- Does not include personal or sensitive information
- Remains on the user’s device
- Is not transmitted externally

## 5. Permissions
The Extension requests minimal permissions required to function:
- Host permissions for reading email/page content **only when the user activates the extension**
- Storage permission for local settings

The Extension does **not** monitor browsing behavior.

## 6. Security
All communication with the backend is encrypted using HTTPS.
